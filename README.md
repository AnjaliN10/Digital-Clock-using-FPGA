
No board-specific constraint files are included to keep the design portable.
# Usage
- The module can be synthesized on any FPGA platform  
- Outputs can be connected to:
  - Seven-segment displays
  - LCD modules
  - UART / simulation waveform viewers  
This repository focuses on core timekeeping logic only.
# Applications
- FPGA learning and practice  
- Digital design coursework  
- Embedded systems fundamentals  
- Resume and portfolio demonstration
